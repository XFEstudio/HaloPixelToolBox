# 🎧 HaloPixelToolBox（花再音响工具箱）

一个面向桌面端的实用工具箱，致力于为音响 / 桌面设备提供更丰富、更直观的信息展示能力。

---

## 🚧 当前开发进度

- ✅ 已支持 **网易云音乐歌词显示**
- 🚀 其它音乐平台歌词正在开发中

> 项目仍处于持续迭代阶段，功能和体验都会不断完善。

---
